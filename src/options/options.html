Forget Specific History Options

In the form below, add the domains of websites you'd like to not be stored in history.

More details...

Use only the domain name and extension. For example, use yahoo.com to block all yahoo.com websites.

Note that the extension takes a domain - with optional www - and matches the main domain with optional www as well as both http and https protocols.

yahoo.com
  • http://yahoo.com
  • https://yahoo.com
  • http://www.yahoo.com
  • https://www.yahoo.com
  • https://www.yahoo.com/some-article-here
